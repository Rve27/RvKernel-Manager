<p>Unlock the true potential of your Android device with RvKernel Manager, a powerful and user-friendly open-source tool for managing your kernel settings. This app provides comprehensive control over your device's kernel parameters.</p>

<p>Fine-tune your device for peak performance, maximum battery life, or a perfect balance between the two.</p>

<h3>Features:</h3>

<h4>Device Information Dashboard</h4>
Monitor critical device information including CPU model, GPU model, RAM, kernel version, Android version, and more.</p>

<h4>Battery Monitoring & Control</h4>
<ul>
  <li><strong>Real-time monitoring</strong>: Voltage, temperature, charging level, and deep sleep time</li>
  <li><strong>Fast charging control</strong>: Enable/disable for supported devices</li>
  <li><strong>Bypass charging mode</strong>: Supported on compatible kernels</li>
  <li><strong>Battery health tracking</strong>: Monitor current health and capacity</li>
  <li><strong>Thermal configuration</strong>: Manage thermal profiles for optimal performance</li>
</ul>

<h4>Advanced CPU & GPU control</h4>
<ul>
  <li><strong>Multi-cluster CPU support</strong>: Little, Big, and Prime core clusters</li>
  <li><strong>Governor selection</strong>: Choose optimal CPU & GPU scheduling policies</li>
  <li><strong>Frequency scaling</strong>: Set custom min/max frequencies</li>
  <li><strong>GPU tuning</strong>: Power level control, Adreno boost, and throttling settings</li>
</ul>

<h4>Kernel Parameter Tuning</h4>
<p>Fine-tune kernel settings for optimal performance:</p>
<ul>
  <li><strong>Memory management</strong>: Swappiness, dirty ratio, and ZRAM configuration</li>
  <li><strong>ZRAM optimization</strong>: Size allocation and compression algorithm selection</li>
  <li><strong>Network tuning</strong>: TCP congestion control algorithm selection</li>
  <li><strong>Scheduling</strong>: Kernel task scheduler tuning with BORE scheduler support</li>
  <li><strong>Kernel profiles</strong>: Pre-configured profiles (Performance, Balance, Powersave)</li>
  <li><strong>CPU tuning</strong>: Uclamp settings for fine-grained CPU performance control</li>
  <li><strong>And many more advanced parameters!</strong></li>
</ul>

<h4>Material 3 Expressive Design</h4>
<ul>
  <li><strong>Modern interface</strong>: Clean, intuitive Material 3 design</li>
  <li><strong>Dynamic theming</strong>: Adapts to your system colors</li>
  <li><strong>Dark/Light modes</strong>: Full theme support</li>
</ul>

<h4>System Integration</h4>
<ul>
  <li><strong>WireGuard support</strong>: Monitor WireGuard kernel module status</li>
  <li><strong>Real-time monitoring</strong>: CPU usage, uptime, and system metrics</li>
</ul>

<h4>Requirements</h4>
<ul>
  <li><strong>ROOT ACCESS REQUIRED</strong>: This app needs root permissions to modify kernel parameters</li>
  <li><strong>Android 12+</strong>: Minimum supported Android version</li>
  <li><strong>Snapdragon devices</strong>: Currently only compatible with Qualcomm Snapdragon SoCs</li>
</ul>
